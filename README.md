# 1. Repository에서 다루는 내용
- 두 개의 HC-05 모듈을 이용하여 서보를 컨트롤 하는 예제를 진행하여 본다. 
- 입력은 유니티를 통해 받도록 한다. 


# 2. 요구사항
A. 입력을 받으면, Slave 모듈에 연결되어 있는 서보가 움직여야 한다. 
B. 서보의 각도값을 얻어 이를 유니티 상에 표시한다. → MASTER 모듈에서는 명령을 내리고, SLAVE 모듈에서는 명령을 받아 서보를 움직이고, 각도를 반환한다.
